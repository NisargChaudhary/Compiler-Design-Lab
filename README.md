This repository contains the complete set of **Compiler Design (CD) Laboratory Practicals** implemented as per the university syllabus.  
The aim of these practicals is to understand the fundamental phases of a compiler, parsing techniques, and lexical analyzer generators using **C** and **LEX/Flex**.

All programs are written in a simple and understandable manner for:
- Lab examinations
- Viva preparation
- Concept clarity in Compiler Design

1. Program to implement **Lexical Analyzer**  
2. Program to **count digits, vowels, and symbols** in C  
3. Program to **validate User Name and Password** in C  
4. Program to implement **Predictive Parsing (LL(1))** in C  
5. Program to implement **Recursive Descent Parsing** in C  
6. Program to implement **Operator Precedence Parsing** in C  
7. Program to implement **LALR Parsing** in C

### Programs using LEX

9.  
   a. Create a Lexer to take input from a text file and **count number of characters, lines, and words**  
   b. Write a Lex program to **count vowels and consonants** in a given input string  

10.  
   a. Write a Lex program to **print all numbers** from the given file  
   b. Write a Lex program to **print all HTML tags** from the file  
   c. Write a Lex program to **add line numbers** to the given file and display output
